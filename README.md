# 🌦 Weather App

A simple and responsive Weather App that allows users to search for any city and get real-time weather information.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡 Shows temperature
- ☁ Displays weather condition
- 📱 Simple and responsive UI

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- Weather API
